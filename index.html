This Website

Inspired by Tunis by celtano, I created this website from scratch using HTML, CSS & JavaScript, tailoring it to my own vision. CSS Grid & Flexbox are used to make the website Responsive. And vanilla-tilt.js library is used to add the tilt effect to photos/project cards.

GitHub Live Site

02

Match Cards Game

Using the MVC Pattern I created this fully Responsive game where the player must find 2 matching cards to win. Only using HTML, CSS & JavaScript. CSS Animations are used for background lighting. Cards are sorted randomly at the beginning of the game. Clicked card is revealed while others hidden.

GitHub Live Site

03

Countdown Clock

I created this fully Responsive countdown clock using HTML, CSS & JavaScript while following the MVC Pattern. User inputs are validated and calculated to fit standard time units. Ex: 65 minutes will automatically convert to 1hr 05mins. Once countdown is completed, an alert message will deploy.
